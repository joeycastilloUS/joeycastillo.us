§2 — a-d-d placement (new spec doc)

Add a new skill at nous/action/skill/pi-vm/. Why action skill?

  • Pi VM is about executing bytecode — that's the action subsystem (cpu, execute, translate live there)
  • Not memory (storage), not comms (transport), not security (auth/crypto). Execution is action.
nous/action/skill/pi-vm/
  CLAUDE.md       — Pi VM overview, intent, "Future" status
