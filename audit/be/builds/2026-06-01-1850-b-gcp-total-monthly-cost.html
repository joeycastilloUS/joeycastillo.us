Cost-cut levers still on the table (your call)

Lever 1 — Release the openvpn static IP ($3.65/mo)

The static external IP 34.82.60.192 is attached to openvpn-1-vm in the acme-vpn project. The VM is STOPPED, so GCP now charges the unused-reservation fee. Action: release the IP — gain $3.65/mo. Cost: VM gets a different IP on resume.

Lever 2 — Resize the openvpn disk down ($1+ /mo)

The openvpn VM has a 20 GB pd-balanced disk ($2/mo). OpenVPN needs ~2-3 GB. Resize to 10 GB pd-standard → ~$0.40/mo. Saves $1.60/mo. Easy if VM stays stopped.

Lever 3 — Migrate acmedev-platform-hq buckets to Coldline or Archive ($15-22/mo)

~1.03 TB across 17 buckets. The "-cold" named buckets are likely already Coldline; the rest are likely Standard. Migrating non-cold ones to Coldline or Archive:

  • Standard → Coldline: $0.020/GB → $0.004/GB = save ~$16/GB-TB-mo → ~$16/mo savings
  • Standard → Archive: $0.020/GB → $0.0012/GB = save ~$19/GB-TB-mo → ~$19/mo savings

Caveat: retrieval cost goes UP (Archive: $0.05/GB to read). Only migrate buckets you genuinely don't read often.

Lever 4 — Delete legacy snapshots ($0.53/mo)

14 historical "-arc" snapshots from 2026-05-09 consolidation. Tiny absolute cost but cosmetic cleanup. Per the prior audit, these were retained as "rollback window for 30 days" — that window has long passed. Safe to delete.

Lever 5 (DESTRUCTIVE) — Delete the 3 stopped VM disks ($13/mo)

Ashburn 50GB pd-SSD ($8.50) + Singapore 20GB pd-std ($0.80) + openvpn 20GB pd-balanced ($2). Total $11.30/mo. Destructive — no resume. Only if you're certain you'll never need those VMs back.
