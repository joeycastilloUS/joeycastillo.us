What It Does

metal is a single-command provisioning system. It strips bloatware, configures the OS, installs a full dev toolchain, and sets up identity — then gets out of the way.

  • Windows — remove bloatware (Cortana, Copilot, McAfee, Xbox), dark mode, privacy, power, taskbar. Install Git, GCC, Python, Node, Rust, gh, gcloud, Claude Code, Wrangler, Snowflake CLI, 4 browsers
  • Mac — Xcode CLI Tools, Homebrew, Git, Node, Claude Code, gh, gcloud. ./go.sh setup
  • Linux — auto-detects apt, dnf, or pacman. Git, GCC, Node, Claude Code, gh, gcloud. ./go.sh setup
  • Identity — git config, Claude settings, project memory — synced on every launch
