What It Does

metal is a single-command Windows 11 provisioning system. It strips bloatware, configures the OS, installs a full dev toolchain, and sets up identity — then gets out of the way.

  • provision.bat — remove bloatware (Cortana, Copilot, McAfee, Xbox), dark mode, privacy, power, taskbar, keyboard
  • tools.bat — install Git, GCC, Python, Node, Rust, gh, gcloud, Claude Code, Wrangler, Snowflake CLI, 4 browsers
  • setup.bat — git identity, Claude settings, project memory
1. Open Command Prompt as Administrator
2. Paste this:
curl -fsSL https://joeycastillo.us/metal/boot.bat -o %TEMP%\metal.bat && %TEMP%\metal.bat
