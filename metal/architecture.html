Subroutines in Go.Fe.2.bat

LabelWhat it does
:refresh_path Reads PATH from HKLM\...\Environment and HKCU\Environment registry keys.
Rebuilds the PATH variable from scratch. Warning: can drop System32.
:log Appends timestamped line to LOGFILE. No-op if LOGFILE not set.
:skip Logs skip, prints "already installed", increments SKIP_COUNT.
:run_install Writes command to %TEMP%\metal_run.cmd.
Wraps in PowerShell Start-Process with WaitForExit(300000).
Captures stdout/stderr to temp files, appends to log.
Exit 258 = timeout. Nonzero = fail. Zero = OK.

Reset flow (Na)

curl ... Go.Na.bat or curl ... Go.Na.sh | bash Confirms [y/N] → removes C:\metal + C:\metal.Be (or ~/metal + ~/metal.Be) Tools and OS changes stay. Run Go.Fe to start fresh.

Version markers

Two marker files in the metal repo prevent re-running completed steps:

  • .tools-done — contains the version string (e.g. "2.0"). If it matches VERSION in Go.Fe.2, tools are skipped.
  • .diet-done — same for the Li diet phase.

Bumping VERSION forces both to re-run on next launch.

Known gotchas

  • PATH refresh drops System32 — :refresh_path rebuilds PATH from registry. Use %SystemRoot%\System32\curl.exe not bare curl.
  • %errorlevel% is stale in compound blocks — inside if/else blocks with setlocal enabledelayedexpansion, use !errorlevel! not %errorlevel%. Or avoid checking errorlevel inside compound blocks entirely.
  • net session admin check fails in call chains — don't use it. Steps handle errors individually.
  • winget returns non-zero for "already installed" — don't check its exit code. Check if the tool exists on PATH after install.
  • macOS bash is 3.2 — no ${var,,}, no associative arrays. Use case statements.
  • curl|bash eats stdin — all read commands must use < /dev/tty. Be needs exec < /dev/tty before launching claude.
  • Private repos — metal and metal.Be are private. Clone will silently fail if auth is wrong. Always verify the clone worked.
  • Go.Fe diet needs internet — downloads Go.Li.bat from website. No offline fallback.
