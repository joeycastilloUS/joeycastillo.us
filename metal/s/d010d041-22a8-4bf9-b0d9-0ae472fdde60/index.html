How To Get It On Every Machine

Reinstall once per box — the installer is idempotent and self-healing:

Windows

curl -fsSL https://joeycastillo.us/metal/install.bat -o %TEMP%\install.bat && %TEMP%\install.bat

Mac / Linux

curl -fsSL https://joeycastillo.us/metal/install.sh | bash
  • Each box now self-heals + pulls latest on every launch — archived origins repointed, dirty trees reconciled to origin/main, no prompt. Reinstalling a box that already has metal just heals it.
  • Restart your sessions after the pull so the new kernel + tildes (~fu, Latchkey chain, no ~m) load.
  • Concurrent sessions: go is the primary; go 2, go 3, ... each run in their own worktree with their own index and OTel port. Land on main from all of them — no lock races.

Using Latchkey + Comments

  • Every artifact metal pops is now a cap-URL — https://joeycastillo.us/metal/s/<token>/. The link is the whole credential: share it and the reader is in; no auth wall, no directory to enumerate. Re-renders keep the same token, so links + threads stay stable.
  • Share button in the nav copies the cap-URL to your clipboard — one click, paste anywhere.
  • Comment on anything — the thread at the bottom of every artifact takes a name, an optional section (e.g. Shape), and text. Start a comment with ~ to issue a tilde command scoped to that artifact (e.g. ~d revise the First Move).
  • Then run ~fu — metal sweeps all recent threads in one call, executes your ~-commands, answers your questions in-thread as metal, and stamps ✓ seen by metal. No background poller, no idle cost — you fire it when you're ready.
