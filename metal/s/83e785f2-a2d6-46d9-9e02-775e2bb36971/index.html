What Shipped

Deep code analysis (4 Fable agents across the whole tool surface)

  • Data loss — autoheal was stashing uncommitted work on every launch and never restoring it. Now stashes only to reconcile, restores after, keeps it safe + loud on conflict, and won't force-move a feature branch. Harness-verified (both .sh + .ps1). b68c12f
  • XSS containment — </script> breakout in board views + the public telemetry page (board text → stored XSS on published pages). Escaped. e3c6792
  • Mac/Linux commits — pre-commit hook was python-only; now resolves python3. b68c12f
  • Voice prompts — capture read stdin as cp1252, dropping dictated smart-quotes; now UTF-8. Collector signal deadlock fixed; telemetry schema drift (page + ~h Latest blank for weeks) normalized. b68c12f e3c6792
  • Launch + Linux install — go.sh no longer aborts on a fresh board; Go.Fe installs nodejs (was node → apt/dnf failed); latchkey validates ledger tokens are real UUIDs. 5a58524

Live-install hotfixes (from running it on the fleet)

  • Go.Sign email FAIL — gh api user hides .email and the user/emails fallback needs a scope the token lacks. Now falls back to the scope-free noreply email <id>+<login>@users.noreply.github.com; completes green. 5622782
  • Install ran stale scripts — a clone that didn't fully reconcile ran an old Go.Fe (MCP-add) / Go.Sign. The installer now git checkout origin/main -- Go.Fe.* Go.Sign.* go.* right before running them, decoupled from autoheal. (site install.sh + install.bat)
  • Site was out of sync — site had no Go.Fe.ps1 / Go.Sign.*; synced so it mirrors metal-console.
  • MCP no longer torches operators — the nuclear remove wiped a peer's personal MCP (playwright/apify/slack/…). Now honors ~/.metal-mcp-ignore (keep-list) and backs up ~/.claude.json before removing. 9f50a38
  • Fleet access — operator granted a peer read access to the private repo; distribution stays private + per-operator (no Ledger exposure).

How To Get It On Every Machine

Windows:     curl -fsSL https://joeycastillo.us/metal/install.bat -o %TEMP%\install.bat && %TEMP%\install.bat
Mac/Linux:   curl -fsSL https://joeycastillo.us/metal/install.sh | bash

Force-refreshes claude-code to latest, force-checks-out current Go.Fe/Go.Sign before running them, self-heals the clone (restoring your local edits, not stashing them away), and launches. Requires read access to the private kastil-systems/metal-console (grant per operator).

Keep your own MCP servers: create ~/.metal-mcp-ignore with one server name per line — metal will never remove those. A backup of ~/.claude.json is written to ~/.claude.json.metal-mcp-backup before any removal.
