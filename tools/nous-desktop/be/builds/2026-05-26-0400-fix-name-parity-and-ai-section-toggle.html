Bug 2 — AI-section OFF row didn't actually mute the fan-out

Joey: "I disable the ON button on the AI line and when I type something in the anything goes box, all AI are used instead of my thinking and main AI."

Symptom: The DATA ACCESS column has section dividers (LOCAL, CATALOG, WEB, AI) each with an ON/OFF button. Clicking AI → OFF visually dimmed the tile column but providers still fired on the next task.

Root cause: applyDapSectionState('AI') at loop.js:1414 only hid the panels visually:

// BEFORE
if (label === 'AI') {
  providers.forEach(function(name) {
    var panel = providerPanels[name];
    if (panel) panel.style.display = dapEnabled['AI'] ? '' : 'none';
  });
}

No set_disabled WS message was sent. Server-side providers stayed enabled, fan-out hit them all.

Fix: When the section toggles, ALSO send set_disabled for every provider in the section + sync the per-tile toggle visual state + update the master toggle indicator:

// AFTER
if (label === 'AI') {
  var isDisabled = !dapEnabled['AI'];
  providers.forEach(function(name) {
    var panel = providerPanels[name];
    if (panel) {
      panel.style.display = dapEnabled['AI'] ? '' : 'none';
      // Sync per-tile toggle button visual state
      var btn = panel.querySelector('.prov-toggle');
      if (btn) {
        btn.classList.toggle('enabled', !isDisabled);
        btn.classList.toggle('disabled', isDisabled);
      }
    }
    disabledProviders[name] = isDisabled;
    // Tell the server to actually disable / re-enable
    if (ws && ws.readyState === 1) {
      ws.send(JSON.stringify({
        type: 'set_disabled',
        provider: name,
        disabled: isDisabled
      }));
    }
  });
  updateMasterToggle();
}

Combined with yesterday's ~f driver-runs-when-fanout-off fix, the operator now gets the right behavior end-to-end:

  • Click AI section OFF → all 30 provider tiles disabled server-side. Fan-out goes silent.
  • Type a question → MAIN AI (claude-opus-4-7 by default) still answers via the THINKING column (operator-picked driver overrides the disabled flag).
  • Honest trace: SENSE line says "driver runs despite master-off (operator pick wins)".
