What This Sets Up

The CLAUDE.md convention block becomes the rule for every future provider-table edit:

  • When OpenAI ships gpt-5.6: change model on the gpt entry; rename gpt-prev → gpt-5.5 reference; the alias stays gpt.
  • When Anthropic ships Opus 4.8: change model on claude-opus from "claude-opus-4-7" to "claude-opus-4-8"; add claude-opus-prev if we want to keep 4-7 reachable.
  • When a new coding-specialist variant ships from any vendor: use -codex (OpenAI convention) or -coder (alternate).
