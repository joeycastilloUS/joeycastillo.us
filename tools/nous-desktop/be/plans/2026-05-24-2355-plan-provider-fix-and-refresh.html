5. Constraints

  • Rule 4 (Joey pastes, Be handles). Anthropic + Mistral keys arrive in chat from Joey, never asked for one at a time; Be runs setx, verifies env, never echoes the value. Keys captured in transcript trigger rotation reminder per SETUP.md convention.
  • No new external dependencies. Pure C, same toolchain (gcc/MSYS). Provider list is a ProviderDefault array literal — no schema changes, no parser changes.
  • Pricing in the table is informational only. Daily-total tracking uses these. If a model's pricing has changed, update; if uncertain, leave as-is (worst case: a stale pricing report — not a correctness bug).
  • MAX_PROVIDERS limit. Currently 22 entries; adding up to 4 brings us to 26. Need to check MAX_PROVIDERS define in serve.c — bump if needed (likely set generously already).
  • Server must restart. Provider table is compile-time. Wwwroot edits don't apply; only a rebuild + relaunch propagates the table change.
  • WDAC launch path is fragile. PowerShell & works; Start-Process doesn't. Same procedure as the auth-bypass build.
