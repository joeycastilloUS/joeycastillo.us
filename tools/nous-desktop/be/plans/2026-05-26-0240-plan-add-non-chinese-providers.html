6. Constraints

  • Naming convention from nous-desktop/CLAUDE.md §Provider Naming applies: unadorned <vendor> = current flagship; variants use -mini, -fast, -codex, etc.
  • Rule 4 — Joey pastes, Be handles for each new key.
  • MAX_PROVIDERS bump first (in idx:1). Otherwise the array fills up before all adds land.
  • Format type discipline: prefer existing openai format. Only add a new format type (cohere?) if absolutely necessary — adds branches everywhere.
  • Probe-first protocol: before committing each entry to g_defaults[], probe the endpoint manually with curl/PowerShell to confirm body shape and response shape. Saves rebuild cycles.
  • Cost ceiling: total new monthly cost ~$10 (MiniMax) plus free tiers everywhere else. No surprise bills.

  5. Per-vendor latency under load — adding 9 tiles to fan-out adds parallel HTTP calls.
