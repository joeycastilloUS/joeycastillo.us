How to read the signal

  • MATCH count near 208 = clone and prod are aligned. Healthy frozen-clone state right after a fresh clone.
  • MATCH count drops sharply = an Airbyte sync run just landed on prod and bulk-UPSERTed many streams.
  • MATCH count slowly drifts down over hours = steady prod writes accumulate while clone stays frozen. What you see above.
  • MATCH count climbs back up = airbyte-philz cron has resumed and clone is catching up.
