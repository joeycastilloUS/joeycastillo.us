Cost reality is even better than the v2 plan estimate. Warm-cache sweeps run in 15-17s, not 27s. At 5-min cadence: 16s × 288/day = 4,608s = 1.28 hours = 1.28 credits/day. At Enterprise ~$3.50/credit: ~$4.50/day = ~$135/month. Way below the $420-500 v2 estimate. The Cloud Run cold-start tax (87-121s for the first sweep) doesn't recur because the WH stays warm at 5-min cadence (auto-suspend = 60s but next tick fires before suspend completes).

How to read the signal

  • MATCH count near 208 = clone and prod are aligned. Healthy frozen-clone state.
  • MATCH count drops sharply = an Airbyte sync run just landed on prod and bulk-UPSERTed many streams. The 205 → 178 jump = one such event.
  • MATCH count slowly drifts down over hours = steady prod writes accumulate while clone stays frozen. Expected when cron is frozen and prod is active.
  • MATCH count climbs back up = airbyte-philz cron has resumed and clone is catching up. Will signal the success of Phase 4 (cron resume) cleanly.
  • Specific streams in DIFFER for long streaks = those streams have ongoing prod changes the clone hasn't seen. Forensic candidates.
